# Getting Started

Add `react-native-flex-layout` to your project in < 1 minute.

## Installation

Open a Terminal in your project's folder and run:

```bash
npm install react-native-flex-layout
```

or with _Yarn_:

```bash
yarn add react-native-flex-layout
```
