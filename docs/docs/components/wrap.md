# Wrap

## Import

```js
import { Wrap } from 'react-native-flex-layout';
```

## Props API

#### `spacing`

The space between each child (even if it wraps).

**Type:** `Spacing`

**Default:** `0`

---

#### `shouldWrapChildren`

If `true`, the children will be wrapped in a `Box` and the `Box` will take the spacing properties.

**Type:** `boolean`

**Default:** `false`
